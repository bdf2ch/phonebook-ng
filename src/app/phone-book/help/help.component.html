Авторизация в приложении В приложении реализована система авторизации пользователей. После прохождения авторизации пользователю становится доступен дополнительный функционал приложения, в том числе: добавление абонентов в избранные и загрузка собственной фотографии на замену текущей.
Для прохождения процедуры авторизации необходимо нажать на кнопку "Авторизация" на панели управления приложения.
Кнопка авторизации пользователя
После нажатия на кнопку откроется диалоговое окно ввода учетной записи и пароля пользователя. Для авторизации в приложении используется учетная запись Active Directory (учетная запись, используемая при входе в Windows).
Диалоговое окно авторизации пользователя


Выбор организации и структурного подразделения Содержимое телефонного справочника структрировано по организациям и структурным подразделениям организаций. В верхнем лувом углу приложения расположена область выбора организации.
Область выбора организации
При нажатии на область выбора организации отобразится всплывающая панель выбора организации.
Панель выбора организации
Нажав на наименование искомой организации, в левой части экрана отобразится иерархическое дерево подразделений выбранной организации. Для закрытия области выбора организации необходимо повторно на нее нажать.
Структурные подразделения выбранной организации
При нажатии на наименование структурного подразделения будет произведена загрузка информации об абонентах, закрепленных за выбранным структурным подразделением, а так же всех дочерних структурных подразделений. Перечень загруженных абонентов будет отображен в правой части приложения.
Перечень абонентов выбранного структурного подразделения


Поиск абонентов В приложении реализован поиск абонентов по ФИО (фамилии, имени, отчеству), а так же по номеру телефона. Для того, чтобы начать поиск, необходимо начать вводить в поле ввода поиска абонентов ФИО искомого абонента или его номер телефона. По мере ввода символов будет производиться поиск абонентов, удовлетворяющих условию поиска. Минимальная длина строки поиска - 3 символа.
Поле ввода поиска абонентов


Просмотр развернутой информации об абоненте При наведении указателя мыши на фотографию абонента поверх фотографии появляется пиктограмма увеличительного стекла.
Пиктограмма увеличительного стекла при наведении указателя мыши
При нажатии на пиктограмму открывается модальное окно с развернутой информацией об абоненте.
Модальное окно с развернутой информацией об абоненте


Маршрутизация АТС В приложении реализована функция маршрутизации АТС. Например: если пользователь, находящийся в "Аппарате управления" ищет абонентов, находящихся в производственном отделении "Северные электрические сети", то телефонный номер искомого абонента будет отображен с учетом кода выхода на АТС искомого абонента.
В любой момент пользователь имеет возможность поменять месторасположение, относительно которого осуществляется поиск. Для этого необходимо нажать на пиктограмму "шестеренки" в поле поиска абонентов, после чего отобразится панель выбора месторасположения осуществляемого поиска.
Панель выбора месторасположения осуществляемого поиска


Избранные абоненты
Функция доступна только в том случае, если пользователь авторизован.
В приложении реализована функция "Избранные абоненты". При наличии у пользователя избранных абонентов, перечень этих абонентов отображается при запуске приложения, а в панели управления приложения отображается кнопка "Избранные абоненты" со значком, содержащим количество избранных абонентов.
Кнопка перехода в раздел избранных абонентов
При нажатии на кнопку "Избранные абоненты" в приложении отобразится перечень избранных абонентов пользователя.
Перечень избранных абонентов пользователя
Хорошей практикой считается занесение в категорию "избранных" тех абонентов, к контактным телефонам которых приходится обращаться наиболее часто, так как это экономит время, затрачиваемое на поиск.


Личный кабинет
Функция доступна только в том случае, если пользователь авторизован.
В случае успешной авторизации пользователю доступен раздел "Личный кабинет". Для перехода в раздел следует нажать кнопку "Личный кабинет" на панели управления приложением.
Кнопка перехода в личный кабинет
